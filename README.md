# Система учета производственного цикла (v1.0)

Реализация складского и производственного учета на платформе "1С:Предприятие 8.3".

## Функционал
* **FIFO**: Партионное списание себестоимости.
* **Контроль**: Защита от отрицательных остатков.
* **Производство**: Учет материалов и трудозатрат.

## Стек
* 1С:Предприятие 8.3
* Git (Clean History)
